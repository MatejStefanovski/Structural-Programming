# Structured Programming – C++ Exercises (FINKI)

This repository contains a collection of **Structured Programming exercises implemented in C** from the course at **FINKI (Faculty of Computer Science & Engineering)**.  
It serves as a compilation of solved problems used for learning, practicing, and preparing for exams.

## Repository Content

The repository includes solutions from various types of course materials, such as:

- **Laboratory exercises**
- **Auditory (class) exercises**
- **Midterm practice problems**
- **Previous exam tasks**

Some directories contain **multiple exercises within the same source file**. In those cases, different solutions are often **commented out** so that individual tasks can be easily enabled or tested when needed.

In particular, the folders **`vezbi1kolokvium`**, **`vezbi2k`**, and **`vezbi2k24`** include several **midterm practice exercises**, where multiple tasks are implemented and separated through commented sections of code.

## Concepts Practiced

The exercises cover the fundamental ideas of structured programming, including:

- Standard input and output
- Conditional logic (`if`, `switch`)
- Iterative control structures (`for`, `while`, `do-while`)
- Function creation and usage
- Working with arrays and matrices
- String manipulation
- Algorithmic problem solving

## Purpose

This repository is intended as:

- A **study resource for students taking the Structured Programming course at FINKI**
- A **practice collection for midterm and exam preparation**
- A **reference for typical problem-solving patterns in C**

All programs are written in **standard C++** and follow the style and structure commonly expected in FINKI course assignments.
